# 🎓 Smart Attendance Management System (SAMS)

## 📌 Project Overview

Smart Attendance Management System (SAMS) is an automated attendance tracking solution developed using Face Recognition and QR Code technology. The system aims to reduce manual attendance processes, improve accuracy, and provide real-time attendance monitoring for educational institutions.

The application enables students to mark attendance securely while allowing administrators to manage records, monitor attendance, and generate reports efficiently.

---

## 🎯 Problem Statement

Traditional attendance systems are often:

- Time-consuming
- Error-prone
- Difficult to manage at scale
- Susceptible to proxy attendance

SAMS addresses these challenges by leveraging computer vision and QR-based verification to automate attendance management.

## 🔄 Project Workflow

### Step 1: Student Registration
Students are registered into the system along with their profile information and facial data.

### Step 2: Face Encoding Generation
The system generates unique facial encodings for registered users.

### Step 3: Attendance Verification
Attendance can be marked using:
- Face Recognition
- QR Code Verification

### Step 4: Data Storage
Attendance records are securely stored in the SQLite database.

### Step 5: Reporting
Administrators can view attendance records and generate reports through the dashboard.
